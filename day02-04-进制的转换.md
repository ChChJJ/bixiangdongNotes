# 进制的相互转换

| 十进制转二进制：                             |
| -------------------------------------------- |
| 原理对十进制数进行除2运算（余数不是0 就是1） |
| 6 的二进制：                                 |
| 6 / 2 = 3……0                                 |
| 3 / 2 = 1……1                                 |
| 1 / 2 = 0……1                                 |
| 余数倒序排列输出：110                        |
|                                              |

| 二进制转十进制                                               |
| ------------------------------------------------------------ |
| 原理：二进制乘以2的过程                                      |
| 以 110 为例：                                                |
| 0 * 2^0 + 1 * 2^1 + 1 * 2^2                                  |
| 0     +     2     +     4     =     6                        |
|                                                              |
| 二进制位数 对应的 十进制数                                   |
| 0        0        1        1        0        1        0        1 |
| 128      64      32     16       8        4        2       1 |
|                                                              |
